# 📝 PERN Todo App

A simple full-stack Todo application built with the **PERN stack (PostgreSQL, Express, React, Node.js)**.  
Users can create, update, delete, and mark tasks as completed.

---

## 🚀 Live Demo
👉 Live Site: **[https://pern-todo-app-client.onrender.com/]**  
👉 API Base URL: **[https://pern-todo-app-eyu5.onrender.com/]**

---

## ✨ Features
- ➕ Add new tasks  
- 📝 Edit existing tasks  
- ❌ Delete tasks  
- ✅ Mark tasks as completed  
- 🔄 Real-time UI updates  
- 📦 Persistent storage using PostgreSQL  

---

## 🛠️ Tech Stack

**Frontend**
- React (Vite)
- Axios
- Tailwind CSS

**Backend**
- Node.js
- Express.js

**Database**
- PostgreSQL
